Predicting the Oscars

Project Description

I want to train neural networks (NNs) to predict the Academy Awards winners given the nominees for this year. I will specifically aim for predicting the "Best Picture" award by creating and using a dataset of previous nominees and winners and how they fared in earlier awards shows (e.g., SAGs, Golden Globes, BAFTA, etc.).

Project Goals

  1. Learn how voting works
  2. Pick data to be used to predict "Best Picture."
  3. Scrape data for nominees and winners and make one dataset.
  4. Predict the "Best Picture" winner for 2021.
